Git Command Notes

Command Detailed Explanation
cd If you type cd and press the Return key—with no directory specified—you’ll go back to your Home folder. (You can also type cd ~ to go there.)
If you type cd /, you’ll go to the root level of your startup disk.
If you type cd .. (that’s two periods), you’ll go to the directory above the one you’re currently in. So if you’re in your home folder and type cd .., you’ll go to your Mac’s /Users folder.
And if you type cd - (hyphen), you’ll go back to the directory you were in before the last time you issued the cd command.
mkdir To create a new folder (directory), use the mkdir command followed by the folder name. For example, to create a folder named "NewFolder," use: mkdir NewFolder.
touch To create a new file, you can use a text editor or the touch command in the terminal. The touch command is often used to create an empty file. For example, to create a file named "newfile.txt," use: touch newfile.txt.
rm To delete or remove a file, use the rm command followed by the file name. Be cautious, as this action is permanent and cannot be easily undone. For example, to remove a file named "file.txt," use: rm file.txt.
rmdir To delete or remove an empty folder, use the rmdir command followed by the folder name. For example, to remove an empty folder named "EmptyFolder," use: rmdir EmptyFolder.
rm -r To remove a folder and its contents recursively, use the rm -r command followed by the folder name. Be extremely careful when using this command, as it permanently deletes the folder and all its contents. For example, to remove a folder named "MyFolder" and all its contents, use: rm -r MyFolder.
mv To move files from one folder to another, use the mv command followed by the source file and the destination folder. For example, to move a file named "file.txt" from the current directory to a folder named "Documents," use: mv file.txt Documents/.
mv To rename a file, use the mv command and provide the current filename as the source and the new filename as the destination. For example, to rename "oldfile.txt" to "newfile.txt," use: mv oldfile.txt newfile.txt.
